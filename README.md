## 📖 Overview

HirePilot is a modern AI-powered career preparation platform designed to help job seekers prepare for interviews and improve their resumes using Generative AI.

The application allows users to:

- 🔍 Analyze resumes using AI
- 📄 Generate ATS-friendly professional resumes
- 🎯 Generate personalized interview questions
- 🤖 Receive AI-powered interview feedback
- 📊 Save interview reports
- 🔐 Secure authentication with JWT
- ☁️ Upload resumes securely

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/hirepilot.git
```

Move into project

```bash
cd hirepilot
```

---

## Backend Setup

```bash
cd backend

npm install
```

Create a `.env` file.

```env
PORT=3000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_gemini_api_key
```

Run backend

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
